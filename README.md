# Pizza WebApp

## Inhalt
- index.html
- sw.js
- manifest.webmanifest
- icons/

## Auf GitHub Pages veröffentlichen
1. Neues Repository anlegen.
2. Alle Dateien aus dieser ZIP ins Repository hochladen.
3. In GitHub unter **Settings > Pages** als Source den Branch **main** und den Root-Ordner **/** wählen.
4. Nach dem Deploy die Seite im Browser öffnen.
5. Auf dem iPhone in Safari öffnen und über **Teilen > Zum Home-Bildschirm** hinzufügen.

## Hinweis
Wenn du später Dateien änderst, kann der Browser wegen des Service Workers cachen. Dann einmal hart neu laden oder die Homescreen-App löschen und neu hinzufügen.
